# How to Run This Project Locally (Minikube)



This document explains how to run the DevOps Interview Task locally on a Minikube Kubernetes cluster.



---

## Prerequisites



Ensure the following tools are installed on your system:



- Docker Desktop (running)

- Minikube

- kubectl

- Terraform (>= 1.5)

- Git



Verify installations:



```bash

- docker --version

- minikube version

- kubectl version --client

- terraform version





##Local steps

&nbsp;

- Start Minikube

- Clone Repo

- git clone https://github.com/GauravDevops15/devops-interview-task.git

- cd devops-interview-task

- Deploy Infra using Terraform

- Verify application

- Access Monitoring

- Access Centralized logging

- Verify Autoscaling(HPA)

- Cleanup (Destroy all resources)
